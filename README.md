# Runas Zawadihub – FlavorPrint Loyalty Platform (Tier‑based MVP)

Runas Zawadihub is a loyalty & rewards system for a non‑alcoholic food & beverage restaurant.  
Built for an investor pitch, it showcases:

- 🎯 **Tier‑based point multipliers** (Bronze → Platinum) from total spend  
- 🤖 **FlavorPrint** pattern detection for personalised offers  
- 🎁 **Category bonuses** (drinks, food, combos)  
- 🏆 **Real‑time tier progression**  
- 💳 **Order & checkout flow** with instant points  
- 📊 **Interactive dashboard**  

**Tech Stack:** React (Vite) + Tailwind • Python FastAPI • PostgreSQL • Docker Compose
